Quick Start

Get started with Aawaaz.AI API in seconds. No API key required for testing!

GET Request Example
{{ request.url_root }}api?text=Hello+World&voice=US+Male

Response

Returns an MP3 audio file (audio/mpeg) that you can play or download directly.

API Endpoint

Base URL
{{ request.url_root }}api

Parameters

text REQUIRED

The text you want to convert to speech (max 5000 characters)

voice REQUIRED

Voice style from the available options (see voice list below)

Code Examples

JavaScript (Fetch API)

JavaScript 
const generateSpeech = async (text, voice) => {
  const url = `{{ request.url_root }}api?text=${encodeURIComponent(text)}&voice=${encodeURIComponent(voice)}`;
  
  try {
    const response = await fetch(url);
    if (response.ok) {
      const audioBlob = await response.blob();
      const audioUrl = URL.createObjectURL(audioBlob);
      
      // Play audio
      const audio = new Audio(audioUrl);
      audio.play();
      
      // Or download
      const a = document.createElement('a');
      a.href = audioUrl;
      a.download = 'speech.mp3';
      a.click();
    }
  } catch (error) {
    console.error('Error:', error);
  }
};

// Usage
generateSpeech('Hello from Aawaaz.AI!', 'US Male');

Python (Requests)

Python 
import requests

def generate_speech(text, voice):
    url = "{{ request.url_root }}api"
    params = {
        'text': text,
        'voice': voice
    }
    
    response = requests.get(url, params=params)
    
    if response.status_code == 200:
        with open('speech.mp3', 'wb') as f:
            f.write(response.content)
        print("Audio saved as speech.mp3")
    else:
        print(f"Error: {response.status_code}")
        print(response.json())

# Usage
generate_speech("Hello from Aawaaz.AI!", "Indian English Female")
